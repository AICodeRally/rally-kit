# Rally Commands

Show students the available commands in a friendly, visual way.

## Instructions

Print this EXACTLY (fill in the team name from `.team-name` if it exists):

---

**Hey [Team Name]! Here's what you can type anytime:**

| Command | What it does |
|---------|-------------|
| `/rally` | Start fresh or pick up where you left off |
| `/build` | Jump straight to building your app |
| `/polish` | Switch to polish mode — better data, cleaner look |
| `/demo` | Generate your 2-minute demo script |
| `/fix` | Something broke? Paste the error after this |
| `/status` | See what you've built and what's left |
| `/brainstorm` | Stuck on ideas? Let's brainstorm together |
| `/reset` | Start completely over (careful — erases your progress!) |
| `/help` | Show this list again |

**Pro tip:** You can also just talk to me normally — these are shortcuts, not requirements!
